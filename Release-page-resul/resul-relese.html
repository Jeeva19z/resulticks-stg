Digipop Integration for Paid Campaign Activation (Indonesia)

Teams can now launch Digipop-powered ad campaigns directly from RESUL, enabling end-to-end paid activation without switching platforms. This allows Indonesian teams to run paid media activation faster and with more operational consistency.

What’s new

  • Digipop appears as a connected system in Data Exchange → Data Ingestion
  • Target Lists can be built using Digipop partner audience attributes (interests, province, home province, etc.)
  • Create Meta, programmatic, video, generic video, and CTV campaigns directly in RESUL
  • Upload creatives, configure impressions, and assign categories within the RESUL UI
  • Generate Smart Links for all Digipop-supported channels
  • Automatic sync: campaigns created in RESUL appear in the Digipop marketing platform
